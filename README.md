# 学生选课管理系统 python1040

## 项目概述


## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

- 学生模块
  - 选课功能
  - 退课功能
  - 查询课表
- 教师模块
  - 发布课程
  - 修改课程信息
  - 查看选课名单
- 管理员模块
  - 课程管理
  - 用户管理
  - 系统设置

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- 操作系统：Linux或Windows
- Python版本：Python 3.6 或以上
- 数据库：MySQL 5.7 或以上

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库，并进行配置
3. 克隆项目源码
4. 导入数据库文件
5. 启动后端服务
6. 打开前端页面
